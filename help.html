The graphics look wrong, or all I see is a blank screen. What's that all about?

Your graphics drivers may need to be upgraded. Go to the website of the manufacturer of your graphics card to find the most recent drivers.

Why is the game running slowly, or why does the mouse cursor seem to be so jerky?

Tropix and Tropix 2 require a 3D video card supporting DirectX 7 in order to run at full speed. Make sure your computer supports the minimum system requirements. If it does, but your game still runs slowly, you may need to update your drivers.

Can multiple users have different profiles (i.e. saved games)?

In Tropix 2, yes! The first dialog you see each time you start Tropix 2 includes the option to switch profiles.

Tropix 1 does not support multiple user profiles, but you can reset your game on the "options" menu. Make sure you have the latest version of Tropix 1 installed to access this feature.

What are the recommended system requirements for Tropix?

In order to run properly, Tropix requires at least the following:

  • Windows XP or Vista
  • 500HMz or faster processor
  • 128MB RAM
  • 3D Video Card
  • DirectX 7.0 or better

What are the recommended system requirements for Tropix 2?

In order to run properly, Tropix 2 requires at least the following:

  • Windows XP, or Vista
  • 500HMz or faster processor
  • 128MB RAM
  • 3D Video Card
  • DirectX 7.0 or better

Is there a version of Tropix or Tropix 2 for the Mac?

Sorry, Tropix and Tropix 2 are currently supported on Windows only. We love Macs and hope to provide games for that platform in the future.

How do I start a new game in Tropix?

In Tropix 2, you can simply create a new user profile (click "switch user" from the main menu).

In Tropix, however, there is no in-game way to start over - you'll have to erase your saved game files to make that happen. Find the "user" folder wherever Tropix is installed (for example, C:\Program Files\Tropix\user) and delete the contents of the "user" folder only. That is, keep the empty user folder there, but delete the files inside it. This will restart your game.

How do I disable the Tropix or Tropix 2 screensaver?

Right-click your Windows desktop, select "Properties", and click the "Screen Saver" tab. Here you can enable or disable the Tropix or Tropix 2 screen savers.
